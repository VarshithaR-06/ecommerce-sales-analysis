# 🛒 E-Commerce Sales Analysis

## 📌 Project Overview
This project analyzes 240 online sales transactions 
from January to August 2024 using Python, SQL, 
and Power BI to uncover key business insights.

---

## 🛠️ Tools Used
- **Python** — Data cleaning and analysis
- **Pandas** — Data manipulation
- **SQL (SQLite)** — Business queries
- **Matplotlib & Seaborn** — Charts
- **Power BI** — Interactive dashboard

---

## 📂 Project Structure
- `Online_Sales_Data.csv` — Raw dataset
- `Online_Sales_Clean.csv` — Cleaned dataset
- `data_cleaning.ipynb` — Cleaning notebook
- `data_analysis.ipynb` — Analysis notebook
- `ecommerce_dashboard.pbix` — Power BI dashboard

## 🔍 Key Insights
- **Electronics** is the highest revenue category
- **North America** leads in regional sales
- **Credit Card** is the most used payment method
- Revenue peaked in **March 2024**

## 📈 Analysis Performed
- Revenue by Product Category
- Monthly Revenue Trend
- Revenue by Region
- Payment Method Split
- Top 10 Products by Revenue
